# GSAP Wizard 

## Overview
**GSAP Wizard** is an engaging animated web experience built using **HTML**, **CSS**, and **GSAP (GreenSock Animation Platform)**.  
It showcases smooth scroll-triggered animations, dynamic motion effects, and creative UI transitions — perfect for demonstrating advanced frontend animation skills.

---

## Features
- Smooth, responsive animations using GSAP  
- Scroll interaction and animation sequencing  
- Modern UI layout with motion focus  
- Works in all modern browsers

---

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **GSAP (GreenSock Animation Platform)**
